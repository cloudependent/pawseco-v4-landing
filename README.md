# PawsEco — Landing Page
This repo hosts the public marketing one-pager for **PawsEco** on **https://pawseco.com.au** via GitHub Pages.

## Quick edit
- Edit `index.html`. Commit to `main`. Pages auto-deploys.

## SEO assets
- `robots.txt` — allows indexing.
- `sitemap.xml` — lists https://pawseco.com.au/
- `site.webmanifest` — PWA metadata (icons can be swapped later).
- `favicon.ico` — placeholder.

## DNS
See `deploy-log.txt` for A/AAAA and CNAME records to set on GoDaddy.
